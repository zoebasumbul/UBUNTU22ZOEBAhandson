## How to know my branch name?
git branch
git branch --show-current

For example:
zoeba@INLP47:~/UBUNTU22ZOEBAhandson$ git branch
* main
zoeba@INLP47:~/UBUNTU22ZOEBAhandson$ git branch --show-current
main
zoeba@INLP47:~/UBUNTU22ZOEBAhandson$ 

## How to push my files to git?
git status
git add . (to add all files); git add filename (for specific file)
git commit -m "Add Docker setup notes and git commands" (-m is for message of commititng/save)
git push origin main
git status
